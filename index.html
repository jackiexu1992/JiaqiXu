Selected work


Physical layer assisted MPTCP




                           

we present MPhyTCP, the first MPTCP path manager that adaptively chooses optimal MPTCP connection setup sequence in wireless networks based on cross-layer information.

We implemented MPTCP based on two subflows in lab. We also use several routers to manipulate the real network system.

             

End-to-end full duplex Using multi-hop relay

We use multi-hop cut-through relays to enable end-to-end full duplex. We first propose loop-back interference cancellation to handle interference management when all the relays transmit simultaneously. Then we design power amplification scheme to solve the problem of signal distortion.

We implemented our testbed on NI PXIe-1082 up to three relay nodes and two source nodes. We show that BiPass can achieve 1.6x median throughput gain over state-of-the-art one-way cut-through systems, and 4.09x gain over the decode-and-forward scheme.

             

8x8 MU-MIMO System Testbed Using the Advanced USRP Platform




      

We setup an advanced SDR-based (Software-defined Radio) 8x8 MIMO system testbed in lab. Our goal is to set up a MIMO system for the future research.

We implemented our MIMO testbed on NI USRP-2920 hardware. By using up to 16 USRP-2920 devices, we combine them to form a phase-coherent antenna array for transmission and reception, which scales from 2x2 to 8x8 antenna configurations (consisting of up to 8 transmitters and 8 receivers).
